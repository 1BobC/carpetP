Anatomy of a function

This is the recipe for defining a Python function:

  • def: the def keyword, telling Python we‚Äôre about to start a function definition
  • a name for the function
  • (: opening parenthesis
  • (optional) the names of one or more arguments, separated with ,
  • (optional) the names and values of one or more default arguments, separated with (,) note: we‚Äôll see these in the next section
  • ) closing parenthesis
  • : a colon

The recipe for function contents:

  • a new line
  • indentation (press tab on your keyboard)
  • one or more lines
  • (optional) a return statement

Arguements and their position when declared

Arguments that have default values are called keyword arguments. They can be overridden.
